TAGS BÁSICAS DO HTML

As tags do HTML são utilizadas para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Tag < html >

Este é o elemento básico estrutural do HTML, tanto que todo documento HTML deve se iniciar e finalizar com essa tag.

O elemento < head > providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
